Sport Analyzer — улучшенная версия
Файлы:
- index.html — интерфейс
- styles.css — стили
- logic.js — логика прогнозов (футбол, хоккей, настольный теннис)

Инструкция:
1. Распакуй архив.
2. Открой index.html в браузере.
3. Выбери вид спорта, введи участников и параметры, нажми «Проанализировать».

Дальше можно:
- Подключить API статистики (Opta, StatsBomb, спортивные API) для автозаполнения.
- Добавить сервер и базу данных для хранения исторических матчей и обучения моделей.
