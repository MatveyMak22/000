Sport Analyzer — Pro UI
Описание:
- Универсальный интерфейс: вводи пары участников в одну строку.
- Система распознаёт спорт при явном указании через запятую (например: "Иванов - Петров, настольный теннис").
- Для настольного тенниса рекомендуется указывать sport explicitly.
- Выдача — красивый человекочитаемый анализ (процентные вероятности, тотал, фора, советы).

Файлы:
- index.html
- styles.css
- app.js

Развёртывание:
1. Распакуй архив.
2. Открой index.html или выложи в репозиторий и включи GitHub Pages.
